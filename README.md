# curso-python-proway-20250614

Repositório com as aulas do curso de Python da Proway. Nesse curso, iremos utilizar a versão [3.12](https://www.python.org/downloads/release/python-31210/) do Python, que podemos baixar [nesse link](https://www.python.org/ftp/python/3.12.10/python-3.12.10-amd64.exe). Durante a instalação, não esqueça de marcar a opção **"Add Python.exe to PATH"**.
